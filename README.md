## AutomatedTest(提供一个平台，用于自动化测试.)
![](http://7xk7ho.com1.z0.glb.clouddn.com/automate.jpg)


##方案

###服务端
- 注册/反注册代理
- 服务器推送任务至代理
- 服务器检测用例的变化并决定代理是否更新
- 服务器主动轮训检测代理控制的用例是否完成，以便决定是否继续推送任务
###代理
- 使用xmlrpcsever提供服务，使得服务器可以推送用例等
- 控制用例的执行等
###用例
- 执行并更新自身状态:
	- 开始
	- 进度
	- 结束
	- 结果

##数据库设计

###挂起的用例

字段|说明
----------|------------
task_id	  |用例所属的任务id
name      |用例
version	  |标示用例执行对象的版本


###正在执行用例

字段|说明
----------|------------
task_id	  |用例所属的任务id
name      |用例
version	  |标示用例执行对象的版本
ip		  |用例所执行的测试机
###任务

字段|说明
----------|------------
task_id   |任务id
version   |标示用例执行对象的版本
case_names|用例名列表
email_flag|任务完成是否发送邮件
finished  |任务是否完成
result    |用例执行结果
